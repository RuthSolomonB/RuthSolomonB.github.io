# Portfolio Website - Ruth Befikadu

A professional, responsive portfolio site showcasing 16 months of co-op experience and software engineering projects.

## Live Site

https://ruthsolomonb.github.io/

## Tech Stack

- HTML
- CSS
- GitHub Pages

## Sections Included:

- **Qualifications:** High-level summary of skills (Languages, Data, Systems, DevOps).
- **Professional Timeline:** Summary of roles at Trimble and BCLC.
- **Project Gallery:** Technical breakdown of Marketplace, Health Monitoring, and IoT systems.
- **Contact:** Quick links to LinkedIn, GitHub, and Email.

## How to Deploy:

1. Ensure `index.html` and `index.css` are in the same directory.
2. Open `index.html` in any browser to view.
3. To host online, upload these files to **GitHub Pages** or **Netlify**.
